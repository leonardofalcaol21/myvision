## 🔭 MyVision — Projeto Base Frontend

Este PR introduz toda a estrutura inicial do frontend da plataforma **MyVision**, uma aplicação de indicadores e dashboards estratégicos.

---

### 📦 Stack utilizada
| Tecnologia | Versão |
|---|---|
| React | 18 |
| TypeScript | 5 |
| Vite | 5 |
| Ant Design | 5 |
| ECharts (echarts-for-react) | 5 |
| React Router | 6 |
| Axios | 1.6 |

### ✨ Funcionalidades incluídas

- **Autenticação** com JWT
- **Roteamento** protegido com PrivateRoute
- **Layout** responsivo com Sidebar e Header
- **Dashboard** com KPI cards, gráficos de linha, gauge e barras
- **Painel Admin** com tabela de usuários
- Página 404 amigável
